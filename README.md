# Sum N Numbers Project

## Documentation

### Features
- Calculate the sum of the first N natural numbers.
- Optimized algorithms for better performance.
- User-friendly interface.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/patil-bhagyesh/sum-n-numbers.git
   ```
2. Navigate into the project directory:
   ```bash
   cd sum-n-numbers
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Usage Examples
- To calculate the sum of the first 10 numbers:
   ```python
   from sum import sum_n
   result = sum_n(10)
   print(result)  # Output: 55
   ```

### Modules Description
- **sum.py**: Contains the main function to calculate the sum of N numbers.
- **utils.py**: Helper functions to support the main functionality.
